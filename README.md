## Configurção

Este projeto usa <b>NestJS</b> e <b>TypeORM</b> para conectar com <b>PostgreSQL</b>.

## Setup

```bash
$ npm install
```

## Compile e rode o projeto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
